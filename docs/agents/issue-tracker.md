# Issue tracker: GitHub

Issues and PRDs for this repo live as GitHub Issues at `Sean-Miningah/autotraka-pro`.

## Conventions

- PRDs and implementation issues are created as GitHub Issues using the `gh` CLI
- PRDs use the title format `PRD: <feature name>`
- Issues are labeled with triage labels (see `triage-labels.md`)
- Use `gh issue create` to create issues, `gh issue list` to find them, `gh issue view <number>` to read them

## When a skill says "publish to the issue tracker"

Use `gh issue create` with appropriate title, body, and labels.

## When a skill says "fetch the relevant ticket"

Use `gh issue view <number>` to read the issue.
